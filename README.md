# ailab

Do-as-I-Do challenge — first-party source code only.

Two components:
- `do-as-i-do/` — hand & object reconstruction + 6-DoF tracking pipeline (Tasks 2/3).
- `HO-Tracker-Baseline-Challenge/` — Sharpa dexhand retarget/train/rollout (Tasks 1/4).

Note: model weights, datasets, outputs, large binary mesh/media assets, and the
vendored upstream forks under `do-as-i-do/reconstruction/modules/` (SAM3, SAM-3D,
HaWoR, Fast-SAM3D, tapnet — each under its own license) are intentionally excluded.
